# Sticky Tweaks
Make all of your stickiest dreams come true with this sticky bomb config mod! Not only can you customize basic stats, but you can even add completely new functionality like allowing sticky bombs to proc other items or being able to stick bombs to terrain!

## Configureable
Customize sticky bombs to your liking!
* Proc chance (default 5%)
* Damage multiplier (default 1.8)
* Proc Coefficient (default 0.5)
* Stick anything! (default false)
